# Персона Декодер

Telegram Mini App для анализа переписок с помощью AI.

## Файлы
- `index.html` — бесплатная версия
- `pro.html` — PRO версия

## Деплой на GitHub Pages
1. Загрузите все файлы в репозиторий
2. Settings → Pages → Branch: main → Save
3. Ссылка: https://ваш-ник.github.io/persona-decoder/

## Подключение к боту
1. @BotFather → /mybots → Bot Settings → Menu Button
2. Вставьте ссылку на index.html

## API ключ
Замените `YOUR_API_KEY` в коде на ваш ключ от Anthropic:
https://console.anthropic.com
